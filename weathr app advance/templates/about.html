Advanced Weather App

Desktop GUI built with Python (Tkinter)

Features

  • City search & “Use My Location” (IP-based)
  • Current conditions with icons
  • Hourly (next 24h) & Daily (5 days) forecasts
  • Unit toggle: Celsius / Fahrenheit
  • Robust error handling

How to Run

  1. Create API key at OpenWeatherMap.
  2. Put it in config.py → API_KEY.
  3. Install deps: pip install -r requirements.txt
  4. Run: python app.py
